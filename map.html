Map

In NewCP heb je een map waar je naar andere plekken kan gaan. Je hebt op deze map in totaal 18 plekken, maar in die plekken zijn weer andere gebouwen waar je in kan gaan.

Content:

  1. Plekken
  2. Games
  3. Shops
  4. Dansen

Plekken

Je hebt 18 plekken op de map, dit zijn de namen van alle plekken waar je heen kan gaan.

  • Beach, je kan vanaf hier naar de lighthouse.
  • Ski Village, hier heb je de Ski Lodge en de Everyday Phoning Facility.
  • Puffle Wild, hier koop je puffles.
  • Ski Hill, op deze berg zijn er 4 verschillende Sled Racing tracks waar je vanaf kan gaan.
  • Dock, hier heb je de game Hydro Hopper.
  • Town, in het dorp heb je de Coffee Shop, Dance Club en Clothes Shop.
  • Welcome, dit is de plek waar je als eerst komt als je inlogt.
  • Snow Forts, hier kan je sneeuwballen gevechten doen met andere penguins.
  • Ice Rink/Stadium, dit is een ijshockey/voetbal stadion.
  • Plaza, bij de Plaza heb je de Pet Shop, Puffle Hotel en Pizza Parlor.
  • Puffle Park, is een speeltuin voor de puffles.
  • Forest, hier heb je niks speciaals.
  • Pond, hier heb je een kampvuur.
  • Dojo Courtyard, er is hier een Dojo.
  • Mine Shack, vanaf hier kan je naar de Mine en naar de School.
  • Skatepark, hier is een skatepark voor de penguins.
  • Cove, hier heb je de game Catchin' Waves, waar je moet surfen.
  • Iceberg, op de grote stuk ijs, kan je de game Aqau Grabber 3000 spelen.

Games

Op deze map staan alle games die je in de game kan spelen. Dit zijn er in totaal 18 op de map (Er zijn nog meer games die niet op de map staan.).

In totaal zijn er iets van 40+ games. Mijn favoriete games zijn Pizzatron 3000, Cart Surfer, PSA Mission en Ice Fishing.

Shops

Op deze map staan alle Shops. Dit zijn winkels voor kleding, meubelen en items. De shops blijven altijd hetzelfde, behalve de kleding winkel. Die update elke maand met nieuwe kleding.

Dansen

Op elke plek van de map kan je als pinguin dansen door op je toetsenboord de letter D te klikken.
